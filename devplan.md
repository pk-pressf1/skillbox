# План поэтапной разработки

### Фаза 0: Discovery & Foundation (2 месяца)

**Цель:** Сформировать техническое и продуктовое видение, заложить архитектурные основы и юридическую базу.

* Определение MVP-контента.
* Архитектурные прототипы ключевых микросервисов.
* Настройка мониторинга и CI/CD пайплайнов.
* Юридическая оценка обработки данных.

### Фаза 1: MVP (4 месяца)

**Цель:** Запустить функциональное ядро для ограниченной аудитории с возможностью сбора обратной связи.

* Базовый профиль + активность + трекинг.
* Интеграция с телефоном и 1–2 типами устройств.
* Простая система тренировок.
* Интеграция с e-commerce (просмотр рекомендаций).
* Базовая геймификация (уровни, награды).

### Фаза 2: Социальные и региональные фичи (3 месяца)

**Цель:** Внедрить социальное взаимодействие и персонализацию по региону для роста вовлечённости.

* Дружба, группы, поиск пользователей.
* Региональные промоакции.
* Улучшенные сравнения (с регионом, с собой).

### Фаза 3: Опытно-промышленная эксплуатация (ОПЭ) (3 месяца)

**Цель:** Проверить стабильность, масштабируемость и бизнес-эффект системы в условиях, приближенных к реальной массовой эксплуатации.

* Запуск платформы для ограниченной публичной аудитории (до 100K пользователей в 10 странах).
* Включение всех ключевых сценариев:
  * Регистрация → тренировка → достижение → промо → покупка.
  * Участие в региональном челлендже.
* Мониторинг:
  * SLA (доступность ≥ 99.5%).
  * Время отклика API (< 300 мс для 95-го перцентиля).
  * Уровень ошибок (< 0.5%).
* Сбор метрик: удержание (D7 ≥ 40%), конверсия в рекомендованные товары.
* Проведение стресс-теста: имитация соревнования с 50K одновременных участников.

### Фаза 4: Масштабирование и персонализация (3+ месяца)

**Цель:** Глобальный запуск, постоянное улучшение через данные и ИИ.

* Развёртывание в 30+ странах, поддержка 10+ млн пользователей.
* ML-модели для персонализации тренировок.
* Расширенная аналитика и A/B-тестирование всех ключевых фич.
* Интеграция с федерациями и тренерами (B2B-модуль).

#### Критически важные компоненты:

* Auth & Privacy Core.
* Activity Tracking Service.
* Social Graph Service (требует раннего load-testing).
